# Space Complexity

**Objective**

In this project, we will learn how to improve the space complexity.

**Problem**

Enhance a function with Improved Space Complexity to O(1).

**Implementation**


```java

public int sum(int[] arr) {
    int sum = 0;
    for (int i = 0; i < arr.length; i++) {
        sum += arr[i];
    }
    return sum;
}
```
