# Space Complexity

**Objective**

In this project, we will learn how to improve the space complexity.

**Problem**

Enhance a function with Improved Space Complexity.

**Implementation**


```java
public class ArraySum {
    public static int sumArray(int[] arr) {
        int sum = 0;
        for (int i = 0; i < arr.length; i++) {
            sum += arr[i];
        }
        return sum;
    }
}
```
